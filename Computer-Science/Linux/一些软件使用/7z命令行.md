
| 功能             | 命令示例                                      | 说明                                      |
|------------------|-----------------------------------------------|-------------------------------------------|
| 压缩文件/目录    | `7z a archive.7z file1 file2`                 | 压缩多个文件                              |
|                  | `7z a archive.7z dir/`                        | 压缩整个目录（包含结构）                  |
|                  | `7z a -tzip archive.zip file`                 | 指定格式为 ZIP                            |
|                  | `7z a -p123456 archive.7z file`               | 带密码压缩（密码直接输入）                |
| 解压文件         | `7z x archive.7z`                             | 解压到当前目录，保留目录结构              |
|                  | `7z x archive.7z -o/path/to/dir`              | 指定解压路径（-o后无空格）                |
|                  | `7z x -p123456 archive.7z`                    | 带密码解压                                |
|                  | `7z e archive.7z`                             | 解压到当前目录，不保留目录结构            |
| 列出内容         | `7z l archive.7z`                             | 查看压缩包内文件列表                      |
| 删除包内文件     | `7z d archive.7z file1 file2`                 | 从压缩包中删除指定文件                    |
| 测试完整性       | `7z t archive.7z`                             | 测试压缩包是否损坏                        |
|                  | `7z t -p123456 archive.7z`                    | 带密码测试完整性                          |
| 分卷压缩         | `7z a -v100m archive.7z large_file`           | 按 100MB 分卷（生成 archive.7z.001 等）   |
|                  | `7z a -v1g archive.7z large_file`             | 按 1GB 分卷（支持 k、m、g）               |

#### 常用单位
- `-v` 后支持：`k`（KB）、`m`（MB）、`g`（GB），如 `-v500k`、`-v2g`

#### 其他实用选项
- 高压缩率：`7z a -mx=9 archive.7z file`（-mx=0~9，9 为最高）
- 自解压档案（Windows）：`7z a -sfx archive.exe file`

提示：密码使用 `-p` 时不显示输入内容，直接在命令后接密码即可。
