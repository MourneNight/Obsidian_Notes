
### NeoTree 文件树浏览器快捷键

#### 全局快捷键（正常模式下）
| 快捷键      | 功能                          |
|-------------|-------------------------------|
| `<leader>e` | 打开/关闭 NeoTree 文件树      |
| `<leader>E` | 打开文件树并聚焦到当前缓冲区文件位置 |

#### 文件树内操作（进入 NeoTree 窗口后）
| 快捷键 | 功能                          |
|--------|-------------------------------|
| `a`    | 添加文件/目录（以 `/` 结尾创建目录） |
| `d`    | 删除选中文件/目录             |
| `r`    | 重命名选中文件/目录           |
| `y`    | 复制选中文件/目录到剪贴板    |
| `p`    | 粘贴剪贴板中的文件/目录       |
| `?`    | 显示 NeoTree 完整帮助菜单     |

#### 常用提示
- 使用 `h` / `l` 或箭头键折叠/展开目录
- `Enter` 或 `o` 打开选中文件/目录
- `.` 显示/隐藏隐藏文件（dotfiles）
- `gg` / `G` 快速跳转到顶部/底部

这些操作均在 NeoTree 窗口内生效，退出窗口后恢复正常编辑模式。
